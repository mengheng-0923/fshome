# 配置 GitHub Secrets
1.导航到仓库设置: 在您的 GitHub * 仓库页面 *，点击 Settings (设置) 标签页。
2.进入 Secrets: 在左侧边栏中，选择 Secrets and variables > Actions
3.添加私钥: 点击 New repository secret (新建仓库 Secret)，创建以下 Secret：
    Name: SSH_PRIVATE_KEY
    Secret: 粘贴您刚才复制的私钥全部内容。
4.添加其他连接信息: 
  同样方式，创建其他需要的 Secret（可选，但在工作流程中很常用）：
    Name: DEPLOY_SERVER_HOST (Value: 您的服务器 IP 地址或域名)
    Name: DEPLOY_SERVER_USERNAME (Value: 您的服务器用户名，例如 root 或 ubuntu)
    Name: DEPLOY_TARGET_DIR (Value: 代码在服务器上的存放路径，例如 /var/www/my-app) 


# 创建 GitHub Actions Workflow 文件
1.创建目录: 在您的项目根目录创建 .github/workflows/ 目录
2.创建 YAML 文件: 在该目录下创建一个 YAML 文件，例如 deploy.yml
